# Enterprise Document Assistant

Multi-tenant **RAG** (Retrieval-Augmented Generation) platform built for production-style interviews:
**Java 17**, **Gradle (Groovy DSL)**, **Spring Boot 3**, **Vavr**, **Hexagonal Architecture**, **CQRS**, **Kafka saga**, **pgvector**, **Redis**, **Ollama**.

## Modules

| Module | Port | Role (CQRS) |
|--------|------|-------------|
| `rag-common` | тАФ | Shared sealed errors, `ResponseMapper`, Vavr types |
| `ingestion-service` | 8081 | **Write** path: upload тЖТ outbox тЖТ Kafka saga |
| `query-service` | 8082 | **Read** path: embed тЖТ retrieve тЖТ LLM answer |

## Prerequisites

- JDK 17+ (for local Gradle dev)
- Docker Desktop (recommended тАФ **full stack runs in Docker**)

## Quick start тАФ everything in Docker (recommended)

```bash
cd enterprise-rag
chmod +x scripts/*.sh
./scripts/docker-up.sh
```

Or step by step:

```bash
docker compose up -d --build
docker compose --profile init run --rm ollama-init   # first time: pull LLM models (~5тАУ15 min)

# Upload a document
curl -X POST http://localhost:8081/api/v1/ingestion/documents \
  -H "X-Tenant-Id: acme-corp" -F "file=@README.md"

# Wait for saga (watch logs)
docker compose logs -f ingestion-service

# Ask a question
curl -X POST http://localhost:8082/api/v1/query/ask \
  -H "Content-Type: application/json" \
  -H "X-Tenant-Id: acme-corp" \
  -d '{"tenantId":"acme-corp","question":"What is this project about?","topK":5}'
```

## Alternative: infra in Docker, apps on host (Gradle)

```bash
docker compose -f docker-compose.infra.yml up -d
docker exec rag-ollama ollama pull nomic-embed-text
docker exec rag-ollama ollama pull llama3.2

./gradlew build
SPRING_PROFILES_ACTIVE=docker ./gradlew :ingestion-service:bootRun   # terminal 1
SPRING_PROFILES_ACTIVE=docker ./gradlew :query-service:bootRun       # terminal 2
```

> **Note:** `docker` profile on the host still expects service hostnames unless you override JDBC/Kafka URLs to `localhost`. For simplest dev, use **full Docker Compose** above.

## JaCoCo code coverage

JaCoCo is **already enabled**. After tests, open the HTML report:

```bash
./gradlew test jacocoAggregatedReport
open build/reports/jacoco/aggregated/html/index.html   # macOS
```

### Formatting (Spotless)

```bash
./gradlew spotlessApply    # format all Java sources (Google Java Format)
./gradlew spotlessCheck    # fail if not formatted (runs with ./gradlew check)
```

### Tests

| Type | What | Command |
|------|------|---------|
| **All** | Unit + integration | `./gradlew test` |
| **Unit only** | Domain/service (mocked) | `./gradlew unitTest` |
| **Integration** | HTTP controllers (`@WebMvcTest`) | `./gradlew integrationTest` |
| **Coverage** | JaCoCo HTML report | `./gradlew test jacocoAggregatedReport` |

See [TESTING.md](TESTING.md) for the full checklist (local, CI, Docker, APIs).

| Command | Purpose |
|---------|---------|
| `jacocoAggregatedReport` | Combined report (all modules) |
